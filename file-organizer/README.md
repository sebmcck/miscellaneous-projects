# File Organizer

A small Python/Tkinter desktop utility that organizes files in a selected folder into subfolders by file extension.

## Features

- Select any folder from a GUI
- Preview changes with dry-run mode
- Move files into extension-based subfolders such as `pdf/`, `jpg/`, `txt/`, and `no_extension/`
- Works from source on macOS, Windows, and Linux where Python/Tkinter is available

## Run from source

```bash
python3 file_organizer.py
```
